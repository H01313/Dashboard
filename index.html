Eerste GitHub-stappen

1. Maak een nieuwe repository in GitHub aan.

2. Voeg precies dit bestand toe met de naam index.html.

3. Klik op Add file en daarna op Upload files.

4. Commit het bestand naar de main-branch.

5. Ga naar Settings → Pages.

6. Kies bij Source: Deploy from a branch.

7. Kies branch main en map /(root).

8. Klik op Save en wacht 1-2 minuten.

9. Open daarna de GitHub Pages link van je dashboard.
